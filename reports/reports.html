PERMUTATION FEATURE IMPORTANCE

One of the most basic questions we might ask, which features have the biggest impact on model predictions?

Permutation Importance plot helps us answer the question.

  • A feature is considered to be important if the model's predictive power drops in case we randomly shuffle it's value for prediction.
  • Similarly model performance may not change or even it may improve if that feature does not have any predictive power .These features can be dropped during model creation.
  • The values towards the top are the most important features, and those towards the bottom matter least.

SHAP FEATURE SUMMARY

  • The summary plot combines feature importance with feature effects.
  • Each point on the summary plot is a Shapley value for a feature and an instance.
  • The position on the y-axis is determined by the feature and on the x-axis by the Shapley value.
  • The color represents the value of the feature from low(blue) to high(red).
  • Overlapping points are jittered in y-axis direction, so we get a sense of the distribution of the Shapley values per feature.
  • SHAP summary plots give us a birds-eye view of feature importance and what is driving it.

PARTIAL DEPENDENCE PLOT

The next questions we might ask, how a particular features affects predictions?

Partial Dependence plot helps us answer the question.

  • The partial dependence plot (short PDP or PD plot) shows the effect a features have on the predicted outcome of a machine learning model keeping the value of all other features fixed.
  • A few items are worth pointing out as you interpret this plot
    • The y axis is interpreted as change in the prediction from what it would be predicted at the baseline or leftmost value.
    • A blue shaded area indicates level of confidence.
  • Knee is the value at which you get the best performance.
    • NOTE - Feature must have atleast two distinct values to find out the Knee points
